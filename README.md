# CodeTime Tracker

**Простой трекер времени для программирования в VS Code.**

## Что делает:
- Считает **активное время** (только когда ты печатаешь/редактируешь код).
- Показывает статистику: **сегодня / неделя / месяц** в строке состояния.
- Сохраняет данные **локально** (без интернета).

## Установка:
1. Установи расширение.
2. Открой любой файл и начни кодить — время пойдёт!

## Команды:
- `CodeTime: Сбросить статистику` (Ctrl + Shift + P).
